# Sobre o projeto
Ignite Teams

Aplicação de criação de turmas e adição de membros nessas turmas.

### Contém as seguintes funcionalidades

- Criar nova turma.
- Adicionar pessoas, tanto na turma A ou B.
- Remover pessoa da turma.
- Remover Turma.

### Tecnologias utilizadas no projeto

- React Native
- Expo
- typescript
- styled-components
- react-navigation
- async storage
